 - provide以及inject
  [文档](https://cn.vuejs.org/v2/api/#provide-inject)

```js
    provide：Object | () => Object
    inject：Array<string> | { [key: string]: string | Symbol | Object }
```

    这对选项需要一起使用，以允许一个祖先组件向其所有子孙后代注入一个依赖，
    不论组件层次有多深，并在起上下游关系成立的时间里始终生效。



 - this.$refs 调用子组件里定义的方法
